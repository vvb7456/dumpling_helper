## 🚀 使用指南

### 1. 安装 Python 3 (Windows)

推荐使用 `winget` 包管理器进行安装。在终端中运行：
```bash
winget install -e --id Python.Python.3.12
```
安装完成后，运行以下命令检查版本：
```bash
python --version
```

### 2. 获取项目文件

**方式一 (推荐): 使用 Git**
```bash
git clone https://github.com/vvb7456/dumpling_helper.git
```
**方式二: 手动下载**

直接从 GitHub 页面下载 ZIP 压缩包，解压后将所有文件放入同一个文件夹。

### 3. 安装依赖

进入项目文件夹，在空白处`右键` -> `在终端中打开`，然后运行：
```bash
pip install -r requirements.txt
```

### 4. 配置群聊列表

在项目文件夹中，创建一个名为 `groups.txt` 的文本文件。在文件中填入需要发送图片的QQ群号或群名，确保**每行一个**，且没有多余的空格或字符。

### 5. 运行脚本

在项目终端中运行：
```bash
python .\dumpling.py
```
脚本会弹出文件选择框，选择图片后，它会倒数5秒。请在这5秒内，**将QQ窗口最大化并置于最前**。

## ⚠️ 注意事项

1.  **兼容性**: 默认支持 `QQ 9.9.20-37625` (深色模式)。如果您的QQ版本或主题不同，请自行截取“发送图片”按钮，命名为 `image_button.png` 并覆盖原文件。

2.  **无人值守**: 脚本运行时会自动操作鼠标和键盘，**请勿进行任何操作**，直至脚本执行完毕。

3.  **识别率**: 为了提高图像识别的准确率，建议将 Windows 系统和 QQ 客户端都设置为**深色模式**。
