# 饺子助手
专为邓老板一键发送饺子匠心研制

## HOW TO USE
1.  **安装 Python 3**

在终端中运行
```bash
winget install -e --id Python.Python.3.12
```

2.  **下载源码**

3.  **安装依赖库**

在项目文件夹右键，打开终端。
```bash
pip install -r requirements.txt
```

4.  **设置变量**

在项目文件夹下新建一个名为groups.txt的文本文档，填入需要批量发送饺子的qq群号（或群名，可能不精确），每行一个，不要有额外字符。

5.  **运行脚本**

```bash
python .\dumpling.py
```
在弹出的选择框中选择需要发送的饺子，然后立刻将qq最大化显示并保持在前台。

## 注意事项

1.默认支持版本：QQ9.9.20-37625（深色模式），其他版本请将发送图片按钮截图并重命名为“image_button.png”，覆盖项目文件夹同名图片。
2.在脚本运行完成之前不要操作键盘鼠标。
3.将Windows改为深色模式会提高图像识别的准确率。
